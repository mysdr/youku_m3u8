Youku m3u8

根据优酷视频ID获取m3u8格式地址，支持清晰度查询，返回不同清晰度视频

Download this project as a .zip file Download this project as a tar.gz file

根据优酷视频id获取视频m3u8地址

能够通过视频id来获取m3u8的地址，该地址可以使用video标签在ios设备中嵌入播放。

  • 支持清晰度获取
  • 支持根据清晰度来获取不同清晰度的视频
